# 🧠 Memory Click Game

A fun and interactive **Memory Matching Game** built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- 4×4 memory grid (16 cards)
- 8 matching pairs
- Smooth 3D flip animations
- Move counter
- Match counter
- Timer system
- Star rating system
- Score calculation
- Restart game option
- Win celebration message
- Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid + 3D Transforms)
- JavaScript (Vanilla)
